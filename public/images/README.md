# Logo KUA

Silakan letakkan file logo KUA di folder ini dengan nama salah satu dari berikut:

- `logo-kua.png`
- `logo-kua.jpg`
- `logo-kua.svg`

Logo akan otomatis muncul di:
1. Header aplikasi (pojok kiri atas, sebelah tulisan "LKH KUA")
2. Halaman login (di atas judul)

**Catatan:**
- Format yang didukung: PNG, JPG, SVG
- Ukuran yang disarankan: 
  - Untuk header: minimal 48x48 pixel (akan di-resize ke 48x48)
  - Untuk login: minimal 80x80 pixel (akan di-resize ke 80x80)
- Jika logo belum diupload, akan ditampilkan placeholder dengan tulisan "KUA"
